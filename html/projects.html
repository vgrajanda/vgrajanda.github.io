Despliegue de aplicación web con AWS

El propósito principal de este proyecto es demostrar la capacidad de desplegar y administrar una aplicación web moderna en un entorno de nube (AWS) utilizando prácticas de DevOps. La aplicación escogida, Spring PetClinic, es una aplicación Java estándar que simula un sistema de información para una clínica veterinaria.
